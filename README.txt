== Description ==

Add a sleek and modern floating menu to your site, allowing you to display social media icons, important links, or any other type of content. This plugin supports four different positions:
- Left Center
- Left Bottom
- Right Center
- Right Bottom

== Installation ==

1. Find and install this plugin in the `Plugins` -> `Add new` section of your `wp-admin`
2. Press `Activate` in the in the `Plugins` list
3. Go to `Breeze Menu` section that will appear in your main menu on the left
4. Click `Add menu item` to create your first menu item
5. Click `Save`
